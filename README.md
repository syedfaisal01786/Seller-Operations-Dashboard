# Seller Operations Dashboard using SQL & Excel

## Tools Used
- Oracle SQL
- Microsoft Excel

## Project Description
Developed a seller operations monitoring system using Oracle SQL and Excel to track seller orders, delayed shipments, and returns.

## Features
- Seller performance analysis
- Delayed shipment tracking
- Return order monitoring
- Excel dashboard visualization

## SQL Concepts Used
- SELECT
- GROUP BY
- ORDER BY
- Aggregate Functions
